   - Choices: A) x = 3 only  B) x = −3 only  C) x = 3 and x = −3  D) x = 9 and x = −9
   - Hint: Vertical asymptotes occur where the denominator equals zero and the numerator does not — factor the denominator and set it equal to zero.
   - Steps:
     - Set denominator = 0: x² − 9 = 0 → (x − 3)(x + 3) = 0.
     - x = 3 or x = −3. Check numerator at these values: neither gives 0.
     - Vertical asymptotes at x = 3 and x = −3.
   - Answer: x = 3 and x = −3

9. **[Hard] Permutations with Repetition**
   - EN: How many distinct arrangements are there of the letters in the word CANADA?
   - FR: Combien d'arrangements distincts y a-t-il pour les lettres du mot CANADA?
   - Choices: A) 60  B) 90  C) 120  D) 180
   - Hint: For arrangements with repeated letters, divide n! by the factorial of each repeated letter count — for AABB: 4! ÷ (2! × 2!) = 6.
   - Steps:
     - CANADA has 6 letters: C, A, N, A, D, A — the letter A repeats 3 times.
     - Number of arrangements = 6! ÷ 3! = 720 ÷ 6 = 120.
     - There are 120 distinct arrangements.
   - Answer: 120

10. **[Hard] Derivative — Optimization**
    - EN: A farmer has 40 m of fencing to enclose a rectangular garden against a barn wall (only 3 sides need fencing). What is the maximum area the garden can have?
    - FR: Un fermier a 40 m de clôture pour entourer un jardin rectangulaire contre un mur de grange (seulement 3 côtés ont besoin de clôture). Quelle est la superficie maximale que le jardin peut avoir?
    - Choices: A) 180 m²  B) 190 m²  C) 200 m²  D) 210 m²
    - Hint: Set up one side as x, write the constraint equation, express area in terms of one variable, then differentiate and set equal to zero.
    - Steps:
      - Let width = x (two widths), length = 40 − 2x. Area A = x(40 − 2x) = 40x − 2x².
      - A′ = 40 − 4x. Set A′ = 0: x = 10.
      - Length = 40 − 20 = 20. Area = 10 × 20 = 200 m².
    - Answer: 200 m²
